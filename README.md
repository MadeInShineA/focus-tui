# focus-tui

A terminal user interface (TUI) application for focus sessions, built with Rust and ratatui.

## Installation

Ensure you have [Rust](https://rust-lang.org/) installed. Then:

```bash
git clone <repository-url>
cd focus-tui
cargo build --release
```

## Usage

Run the application:

```bash
cargo run
```

Or with the built binary:

```bash
./target/release/focus-tui
```
